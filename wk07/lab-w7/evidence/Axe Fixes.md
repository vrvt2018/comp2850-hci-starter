**Critical**
- Alternative text for image

  before:
  ```html
  <img src="/static/img/icon.png" width="16" height="16">
  ```

  after:
  ```html
  <img src="/static/img/icon.png" width="16" height="16" alt="Webpage icon">
  ```


**Serious**
- Elements must meet minimum colour contrast ratio thresholds


- Hyperlinks must be easily visible to click
  
  before:
  ```html
  <a href="/about"></a>
  ```

  after:
  ```html
  <a href="/about">About</a>
  ```
